# blog
关于go的增删改查（带分页和富文本）

Function introduction：
这个项目是对于文章的增删改查的demo，
&ensp;&ensp;增加功能：里边用到的是富文本，富文本里边支持图片上传，动图上传，富文本格式都支持。
&ensp;&ensp;修改功能：里边也有富文本，支持增删内容
&ensp;&ensp;列表页面：列表页面是用的layui插件，支持分页。
&ensp;&ensp;删除就不用说了。
